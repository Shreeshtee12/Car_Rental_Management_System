# Environment and Tool Requirements

## Car Rental Management System – Bonza Cars

To successfully run and develop the **Bonza Cars Car Rental Management System**, the following software environment and tools must be installed.

---

# Required Development Environment

### Operating System

The project can run on the following operating systems:

* Windows 10 or later
* macOS
* Linux

---

# Required Software

### Java Development Kit (JDK)

The application is developed using Java.

Recommended version:

* **JDK 17 or later**

Download from:
https://www.oracle.com/java/technologies/javase-downloads.html

---

### JavaFX SDK

JavaFX is used to build the graphical user interface.

Recommended version:

* **JavaFX SDK 17 or later**

Download from:
https://openjfx.io

---

### MySQL Database

The application stores all system data in a MySQL relational database.

Recommended version:

* **MySQL Server 8.0**

Download from:
https://dev.mysql.com/downloads/mysql/

---

### MySQL Workbench

Used for managing the database and executing SQL queries.

Download from:
https://dev.mysql.com/downloads/workbench/

---

### Integrated Development Environment (IDE)

Recommended IDE:

* **IntelliJ IDEA**

Alternative IDEs:

* Eclipse
* NetBeans

Download IntelliJ IDEA:
https://www.jetbrains.com/idea/

---

# Additional Tools

### GitHub

Used for project repository hosting and version management.

https://github.com

---

### Scene Builder

JavaFX Scene Builder helps design the graphical interface visually.

Download from:
https://gluonhq.com/products/scene-builder/

---

# Minimum Hardware Requirements

* Processor: Intel i5 or equivalent
* RAM: 8 GB recommended
* Storage: At least 2 GB free space

---

# Summary

To run the project successfully, install the following core tools:

* Java JDK
* JavaFX SDK
* MySQL Server
* IntelliJ IDEA
* Git
* Scene Builder (optional but recommended)
